# Sudoku Solver 1.0
This is a simple graphic tool for automatic solving of Sudoku

## Requirments
+ Python3 (any version)
+ PySimpleGUI (installable via pip)

## Usage
```
python3 ./main.py
```
Solver logs his "thinking" to file ***log.txt*** in current directory
